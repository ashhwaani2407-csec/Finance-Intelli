# Finance-Intelli
Its the Intelligence system which helps to analyze the tickers from different asset classes based on parameters like technicals, sentiments, Geopolitical situation, news, and many more with recommendation of Buy/ Sell/ Hold backed with Confidence Score
It gives the recommendation with the Reasoning Summary:- (Sample)
MACD histogram is positive (momentum supportive).
News sentiment is neutral to mixed.
Fundamentals are neutral or missing.
GPR(mock) index=26/100
Decision source: XGBoost model.

Uses XGBoost Model for Calculating the sentiments and news.
